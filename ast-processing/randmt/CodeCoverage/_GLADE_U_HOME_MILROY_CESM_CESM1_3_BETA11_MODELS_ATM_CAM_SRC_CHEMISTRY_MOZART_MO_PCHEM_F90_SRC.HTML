    13) !-----------------------------------------------------------------------------
    14) !   purpose:                                                                 
    15) !   load various "weighting functions" (products of cross section and        
    16) !   quantum yield at each altitude and for wavelength).  the altitude        
    17) !   dependence is necessary to ensure the consideration of pressure and      
    18) !   temperature dependence of the cross sections or quantum yields.          
    19) !   the actual reading, evaluation and interpolation is done is separate     
    20) !   subroutines for ease of management and manipulation.  please refer to

    22) !-----------------------------------------------------------------------------
    23) !   parameters:                                                              
    24) !   nw     - integer, number of specified intervals + 1 in working        (i)
    25) !            wavelength grid

    27) !            working wavelength grid                                         
    28) !   nz     - integer, number of altitude levels in working altitude grid  (i)
    29) !   tlev   - real(r8), temperature (k) at each specified altitude level       (i)
    30) !   airlev - real(r8), air density (molec/cc) at each altitude level          (i)
    31) !   j      - integer, counter for number of weighting functions defined  (io)
    32) !   sq     - real(r8), cross section x quantum yield (cm^2) for each          (o)
    33) !            photolysis reaction defined, at each defined wavelength and     
    34) !            at each defined altitude level                                  
    35) !   jlabel - character*40, string identifier for each photolysis reaction (o)
    36) !            defined                                                         
    37) !-----------------------------------------------------------------------------

    72) !-----------------------------------------------------------------------------

    74)             case( 'jn2o5' )

    79)             case( 'jn2o' )
