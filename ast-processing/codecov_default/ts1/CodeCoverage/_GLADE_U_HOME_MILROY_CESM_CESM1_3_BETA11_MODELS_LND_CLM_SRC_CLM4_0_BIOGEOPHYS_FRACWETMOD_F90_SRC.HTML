    35) ! !DESCRIPTION:
    36) ! Determine fraction of vegetated surfaces which are wet and
    37) ! fraction of elai which is dry. The variable ``fwet'' is the
    38) ! fraction of all vegetation surfaces which are wet including
    39) ! stem area which contribute to evaporation. The variable ``fdry''
    40) ! is the fraction of elai which is dry because only leaves
