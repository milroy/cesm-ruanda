    63) ! Modifications:
    64) !
    65) ! Original: H. Barker

    71) !           if routine cldprop is used; delta scaling can be applied by swithcing
    72) !           code below if cldprop is not used to get cloud properties. 
    73) !           AER, Jan 2005
    74) ! Revision: Modified to use McICA: MJIacono, AER, Nov 2005
    75) ! Revision: Uniform formatting for RRTMG: MJIacono, AER, Jul 2006
